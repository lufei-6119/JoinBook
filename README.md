# JoinBook

## Requirements

### Server Software

* Paper 1.20+ (recommended)
* Compatible with modern Paper-based servers

### Java Version

* Java 17 or higher

## Installation

1. Download the latest version of JoinBook.
2. Place the plugin `.jar` file into your server's `plugins` folder.
3. Start or restart your server.
4. Open the generated configuration file.
5. Customize the book title, author, pages, and other settings.
6. Reload or restart the server to apply your changes.

## Configuration

JoinBook is designed to be simple and easy to configure.

The configuration file allows you to customize:

* Book title
* Book author
* Book pages
* Welcome messages
* Starter-kit notifications
* Other plugin behavior options

No programming knowledge is required.

## Commands

This plugin currently does not provide any player commands.

All functionality is handled automatically when a player joins the server for the first time.

## Permissions

This plugin currently does not require any special permissions.

Server administrators can configure the plugin through the configuration file.

## Compatibility

JoinBook works well alongside many popular Paper plugins, including:

* EssentialsX
* LuckPerms
* WorldEdit
* WorldGuard
* PlaceholderAPI
* Other common Paper plugins

## Why Use JoinBook?

Many servers struggle to communicate important information to new players. Chat messages are often skipped, and players may miss critical rules or instructions.

JoinBook solves this problem by presenting information in a clean and interactive book interface immediately when a player joins for the first time. This creates a more professional onboarding experience and helps players understand your server from the very beginning.

Whether you run a small private SMP or a large public community server, JoinBook provides a simple and effective way to welcome new players.
